Słownik pojęć. Krótkie, uczciwe definicje terminów, które wracają w całym kompendium. Bez żargonu dla żargonu. Gdzie jest pełny dział, prowadzi odnośnik.

{% assign P = site.posts %}
Alfa i beta
Beta to część zwrotu, która wynika z ekspozycji na rynek (ruszasz się razem z nim). Alfa to nadwyżka ponad to, czego nie tłumaczy rynek ani znane czynniki. Większość rzekomej alfy okazuje się nierozpoznaną betą. {% assign p = P | where: "slug", "capm-i-faktory-skad-zwrot" | first %}{% if p %}CAPM i czynniki{% endif %}
Arbitraż
Zysk bez ryzyka z rozbieżności cen tego samego lub równoważnego aktywa. W praktyce niemal zawsze jest to arbitraż z ograniczeniami (koszty, kapitał, ryzyko domknięcia), a nie darmowy obiad.
Autokorelacja
Zależność szeregu od własnej przeszłości. Zwroty są prawie nieautoskorelowane, ale ich kwadraty już nie, co widać jako grupowanie zmienności. {% assign p = P | where: "slug", "szeregi-czasowe-arima-garch" | first %}{% if p %}Szeregi czasowe{% endif %}
Backtest
Symulacja strategii na danych historycznych. Warunek konieczny, nigdy wystarczający: przy dość próbach świetny backtest jest gwarantowany nawet dla czystego szumu. {% assign p = P | where: "slug", "backtest-w-pythonie-szkielet" | first %}{% if p %}Backtest w Pythonie{% endif %}
Bootstrap
Szacowanie rozkładu statystyki przez losowanie ze zwracaniem z własnej próby, bez założeń o rozkładzie. Dla danych zależnych w czasie stosuje się wersje blokowe. {% assign p = P | where: "slug", "bootstrap-i-testy-istotnosci" | first %}{% if p %}Bootstrap i testy{% endif %}
Carry
Zysk z samego utrzymywania pozycji, gdy nic się nie dzieje (np. dodatnia różnica stóp w FX). Znany z tego, że zbiera drobne premie latami i oddaje je w jednym krachu. {% assign p = P | where: "slug", "carry-crash-kiedy-carry-trade-sie-zawala" | first %}{% if p %}Carry i crash{% endif %}
Contango i backwardation
Kształt struktury terminowej. Contango: kontrakt droższy niż spot (rolowanie kosztuje). Backwardation: tańszy (rolowanie dopłaca). {% assign p = P | where: "slug", "kontrakty-terminowe-contango-backwardation" | first %}{% if p %}Kontrakty terminowe{% endif %}
Deflated Sharpe Ratio
Poprawiony wskaźnik Sharpe uwzględniający liczbę wypróbowanych strategii i nienormalność zwrotów. Odpowiada na pytanie, ile z wyniku to umiejętność, a ile szczęście z wielu prób. {% assign p = P | where: "slug", "dlaczego-twoj-sharpe-klamie-deflated-sharpe" | first %}{% if p %}Czemu Sharpe kłamie{% endif %}
Drawdown (obsunięcie)
Spadek kapitału od ostatniego szczytu. Boli asymetrycznie: strata 50% wymaga zysku 100%, żeby wrócić do zera. {% assign p = P | where: "slug", "drawdown-matematyka-obsuniecia" | first %}{% if p %}Matematyka obsunięcia{% endif %}
Dźwignia
Kontrolowanie pozycji większej niż kapitał. Powiększa zysk i stratę symetrycznie, a przy automacie grozi wezwaniem do uzupełnienia depozytu i szybkim zerem.
Edge (przewaga)
Statystyczna nadwyżka oczekiwana ponad koszty. Realna dopiero po odjęciu spreadu, prowizji i poślizgu, i tylko jeśli przetrwa poza próbą. Sam pomysł nie jest przewagą.
EMH (hipoteza rynku efektywnego)
Teza, że ceny odzwierciedlają dostępną informację, więc trwałe bicie rynku jest trudne. Nie znaczy, że rynek ma zawsze rację, tylko że łatwej przewagi nie ma. {% assign p = P | where: "slug", "hipoteza-rynku-efektywnego-najwiekszy-spor" | first %}{% if p %}Spór o EMH{% endif %}
Expected Shortfall (ES)
Średnia strata w najgorszych scenariuszach ponad próg VaR. Lepsza od VaR, bo mówi, jak źle jest w ogonie, a nie tylko gdzie ogon się zaczyna. {% assign p = P | where: "slug", "miary-ryzyka-var-expected-shortfall" | first %}{% if p %}Miary ryzyka{% endif %}
Grube ogony
Rzadkie zdarzenia ekstremalne są znacznie częstsze, niż przewiduje rozkład normalny. Dlatego modele oparte na normalności zaniżają ryzyko krachu. {% assign p = P | where: "slug", "stylizowane-fakty-rynkow-grube-ogony" | first %}{% if p %}Stylizowane fakty{% endif %}
HMM (ukryty model Markowa)
Model z nieobserwowanymi stanami (np. spokój, burza) generującymi obserwacje. Narzędzie kontekstu, nie sygnał wejścia. {% assign p = P | where: "slug", "lancuchy-markowa-i-hmm" | first %}{% if p %}Łańcuchy Markowa i HMM{% endif %}
Kelly (kryterium)
Wzór na wielkość pozycji maksymalizującą długoterminowe tempo wzrostu kapitału. Pełny Kelly jest zwykle za agresywny, stąd praktyka ułamka Kelly. {% assign p = P | where: "slug", "ile-powinienes-grac-kelly-i-sizing" | first %}{% if p %}Kelly i sizing{% endif %}
Kointegracja
Sytuacja, gdy dwa niestacjonarne szeregi mają stacjonarną kombinację liniową, czyli spread wracający do średniej. Fundament pairs tradingu, bywa niestabilny w czasie. {% assign p = P | where: "slug", "stacjonarnosc-i-kointegracja" | first %}{% if p %}Stacjonarność i kointegracja{% endif %}
Kurtoza
Miara grubości ogonów rozkładu. Wysoka kurtoza znaczy więcej ekstremów, niż sugeruje sama zmienność. {% assign p = P | where: "slug", "prawdopodobienstwo-i-rozklady" | first %}{% if p %}Prawdopodobieństwo i rozkłady{% endif %}
Look-ahead bias
Użycie w decyzji informacji, która w danym momencie nie była jeszcze dostępna. Najczęstszy sposób, w jaki backtest sam siebie oszukuje. {% assign p = P | where: "slug", "look-ahead-w-kodzie-realizm" | first %}{% if p %}Look-ahead w kodzie{% endif %}
Maker i taker
Maker dostarcza płynność (zlecenie z limitem czeka w księdze), taker ją zabiera (zlecenie rynkowe). Giełdy różnicują opłaty między nimi. {% assign p = P | where: "slug", "kolejka-zlecen-priorytet-cena-czas" | first %}{% if p %}Kolejka zleceń{% endif %}
Martyngał
Proces, w którym najlepszą prognozą jutra jest dziś. Jeśli cena jest martyngałem, prosta przewaga kierunkowa nie istnieje. {% assign p = P | where: "slug", "procesy-stochastyczne-random-walk" | first %}{% if p %}Procesy stochastyczne{% endif %}
Mikrostruktura
Badanie tego, jak faktycznie powstaje cena: księga zleceń, spread, przepływ, koszty wykonania. Poziom, na którym ginie większość przewag detalicznych. {% assign p = P | where: "slug", "jak-dziala-ksiega-zlecen" | first %}{% if p %}Księga zleceń{% endif %}
Order flow (przepływ zleceń)
Strumień kupna i sprzedaży, który realnie przesuwa cenę. Zagregowany bywa kontekstem, ale u detalisty rzadko daje przewagę po kosztach. {% assign p = P | where: "slug", "order-flow-jak-przeplyw-rusza-kurs" | first %}{% if p %}Order flow{% endif %}
Overfitting (przeuczenie)
Dopasowanie modelu lub parametrów do przypadkowego szumu w danych zamiast do trwałej struktury. Objawia się świetnym wynikiem w próbie i rozpadem poza nią. {% assign p = P | where: "slug", "data-snooping-jak-finanse-sie-oszukuja" | first %}{% if p %}Data snooping{% endif %}
PCA (analiza głównych składowych)
Obrót danych do osi nieskorelowanych, uporządkowanych od największej zmienności. Redukuje wymiar i wydobywa wspólne czynniki, ale bywa niestabilna. {% assign p = P | where: "slug", "algebra-liniowa-pca-w-quant" | first %}{% if p %}Algebra liniowa i PCA{% endif %}
Poślizg (slippage)
Różnica między ceną oczekiwaną a uzyskaną. Rośnie przy niskiej płynności i szybkim rynku, i potrafi zjeść cały teoretyczny zysk. {% assign p = P | where: "slug", "ile-naprawde-kosztuje-twoj-trade-tca" | first %}{% if p %}Koszty transakcji{% endif %}
Reżim rynku
Utrzymujący się stan o określonej charakterystyce (np. niska zmienność, trend, panika). Rozpoznawalny z opóźnieniem, przydatny jako kontekst, nie jako sygnał. {% assign p = P | where: "slug", "jak-rozpoznac-rezim-rynku" | first %}{% if p %}Reżim rynku{% endif %}
Sharpe (wskaźnik)
Zwrot ponad stopę wolną od ryzyka na jednostkę zmienności. Popularny i łatwy do zawyżenia przez wybór najlepszej z wielu prób. {% assign p = P | where: "slug", "dlaczego-twoj-sharpe-klamie-deflated-sharpe" | first %}{% if p %}Deflated Sharpe{% endif %}
Spread
Różnica między ceną kupna a sprzedaży. Pierwszy i najbardziej pewny koszt każdej transakcji, płacony przy wejściu i wyjściu. {% assign p = P | where: "slug", "ile-naprawde-kosztuje-twoj-trade-tca" | first %}{% if p %}TCA{% endif %}
TCA (analiza kosztów transakcji)
Rozłożenie realnego kosztu handlu na spread, prowizję, wpływ na rynek i poślizg. Bez niej brutto myli się z netto. {% assign p = P | where: "slug", "ile-naprawde-kosztuje-twoj-trade-tca" | first %}{% if p %}Ile kosztuje trade{% endif %}
VaR (wartość zagrożona)
Próg straty, którego z zadanym prawdopodobieństwem nie przekroczysz w danym okresie. Mówi, gdzie zaczyna się ogon, nie jak głęboki jest. {% assign p = P | where: "slug", "miary-ryzyka-var-expected-shortfall" | first %}{% if p %}VaR i ES{% endif %}
Walk-forward
Walidacja, w której stroisz parametry na jednym oknie, testujesz na kolejnym i przesuwasz się w czasie. Uczciwsza niż optymalizacja na całej historii. {% assign p = P | where: "slug", "optymalizacja-walk-forward-w-kodzie" | first %}{% if p %}Walk-forward{% endif %}
Zmienność i VRP
Zmienność to skala wahań. Premia za ryzyko zmienności (VRP) to nadwyżka zmienności implikowanej z opcji nad tą, która się realizuje. {% assign p = P | where: "slug", "zmiennosc-prognoza-premia-ryzyka" | first %}{% if p %}Zmienność{% endif %}
